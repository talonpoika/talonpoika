### 🤖 How I build

I work **agent-first** with Claude Code — AI writes most of the code, I drive the architecture, review, and security. Most repos here carry a `CLAUDE.md` so the agent (and you) know the rules of the codebase.

- 🧭 **Plan first** — a planning agent drafts the approach before any code is written
- 🧪 **TDD** — tests first, then make them green
- 🔍 **Automated review** — code + security review pass before every commit
- ⚙️ **Glue everything** — heavy PowerShell / Bash automation, the same instinct I bring to SecOps
